# Reporting and Compliance Matrix

## Core operational reports
- Trial balance
- General ledger
- Journal listing
- AR aging
- AP aging
- Customer statement
- Vendor statement
- Cash book
- Bank book
- Bank reconciliation report
- Inventory movement
- Inventory valuation
- Stock card
- Fixed asset register
- Depreciation schedule

## Tax and invoice outputs
- Sales VAT summary
- Purchase VAT summary
- Tax code usage report
- e-invoice issuance log
- e-invoice error log
- declaration data export set

## Financial statements
- Balance sheet / statement of financial position
- Income statement
- Cash flow statement
- Notes to financial statements
- Closing entries summary
- Retained earnings transfer summary

## Compliance/control requirements
- Full audit trail
- Posted entry immutability
- Reversal instead of destructive edit
- Period locking
- Approval workflow
- Attachment retention
- User and role accountability
- Report reproducibility by period and parameters
- Integration logs for external compliance services
